🤖 Bot Discord + Twitch

// guide d'installation complet

1
🟦 Créer le Bot Discord discord.com/developers
🌐 Va sur discord.com/developers/applications → New Application → donne un nom au bot
🤖 Dans le menu Bot → clique Reset Token → copie le DISCORD_TOKEN
🔑 Dans General Information → copie l'Application ID = ton CLIENT_ID
✅ Dans Bot → active Server Members Intent et Message Content Intent
📨 Dans OAuth2 → URL Generator → sélectionne bot + applications.commands → permissions : Administrator → copie l'URL et invite le bot sur ton serveur
🆔 Dans Discord, active le Mode développeur (Paramètres → Avancés) → clique droit sur ton serveur → Copier l'identifiant = ton GUILD_ID
2
🟣 Obtenir les clés Twitch API dev.twitch.tv
🌐 Va sur dev.twitch.tv/console → Register Your Application
📝 Nom : MonBotDiscord — OAuth Redirect : http://localhost — Category : Chat Bot
🔑 Clique sur Manage → copie le Client ID et génère un New Secret = ton Client Secret
3
📦 Installation et configuration
📁 Extrais les fichiers du bot dans un dossier, puis installe les dépendances :
cd discord-bot
npm install
added 63 packages in 4s
cp .env.example .env
✏️ Ouvre .env et remplis toutes les valeurs avec tes IDs copiés aux étapes 1 et 2
🆔 Pour les IDs des salons et rôles Discord → clique droit dessus dans Discord → Copier l'identifiant
# Discord
DISCORD_TOKEN=MTI3...
CLIENT_ID=11234567890
GUILD_ID=98765432100
# Twitch
TWITCH_CLIENT_ID=abc123def456
TWITCH_CLIENT_SECRET=xyz789...
TWITCH_USERNAME=ton_pseudo
4
🚀 Lancer le bot
node index.js
✅ Connecté : MonBot#1234
✅ Commandes slash enregistrées
🟣 Poller Twitch démarré
💡 Pour le laisser tourner 24h/24 en production, utilise PM2 : npm i -g pm2 && pm2 start index.js --name monbot
